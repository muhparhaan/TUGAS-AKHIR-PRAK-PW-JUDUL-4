Project ini merupakan tugas akhir dari Modul 4 Praktikum Pemrograman Web.  
Tujuan tugas ini adalah membuat aplikasi web sederhana yang dapat digunakan untuk mengelola data kontak.  

Data kontak disimpan menggunakan **session**

---

## âœ¨ Fitur yang Dibuat

### 1. **Login menggunakan Session**
User hanya bisa mengakses halaman utama jika sudah login.  
Session digunakan untuk menyimpan status login.

### 2. **Tampilan Daftar Kontak**
Menampilkan seluruh kontak yang sudah ditambahkan pada session.

### 3. **Tambah Kontak (dengan validasi)**
Form tambah kontak memiliki validasi:
- Semua field wajib diisi
- Email harus valid

### 4. **Edit Kontak**
User dapat mengubah data kontak.

### 5. **Hapus Kontak**
Menghapus data kontak berdasarkan index array.

### 6. **Logout**
Menghapus session dan mengembalikan user ke halaman login.
